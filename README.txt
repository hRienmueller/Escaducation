The Project requires Unity 5.3.0f4. No additional tools or programs are needed.

The following notices are for the Game which can be located in the FTP server.
The game runs best in 1600 x 900 and windowed turned on.
No gamepads or additional input devices are needed as the game functions with the keyboard.
WASD are used for movement. While holding SPACE your character sneaks and does not produce sounds.
If you have an extra collected you may use it through a press of Q.
